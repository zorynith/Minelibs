# **携带版0.1**

**v0.1.0 alpha**是携带版首个公开发布的版本，其于2011年8月16日专门为Xperia PLAY游戏主机发布。

原本计划v0.1.0 alpha在Android平台上发布，但其被推迟了。该版本具有黑色半透明的方向键，并且背景菜单看起来与v0.1.1 alpha大致相同。

### **世界生成**
**生物群系**
- 共有冰层、山丘、海洋和沙漠四种。

**树木**
- 加入了橡树、白桦树和云杉。

**矿团**

### **常规**
- 随机世界
- 在世界的边界会有一层隐形基岩阻挡玩家前进，无限延伸，使用方块搭到世界限高可以到达它的上表面并在那里行走。
- 放置和摧毁方块。
- 自动跳跃
- 虚空
  - 坠入虚空不会死亡，且会无限下坠。
- <span style="color:maroon">版本独有</span>：触摸屏控制与Xperia PLAY手柄控制。
- 邀请无线局域网内的玩家和多人游戏。
- 在暂停菜单中使用“Copy World”按钮能将多人游戏世界保存在设备中。
- 在Minecraft Assets文件夹中一个名为`minecraftpe.apk`的文件能够在Android系统中运行。
- 为高配置和低配置手机准备的选项（分别称为Fancy Graphics和Lower Graphics Quality）。
- Android版加入了Google Play正版验证，未通过验证者不能进入游戏。

携带版的首个演示版本

v0.1.0 alpha的演示版在Xperia Play上发布，发布日期未知。其原型版本也在2011年8月17日在asset.minecraft.net上意外发布，标签为“`mcpe01_canada.apk`”。

### **新内容**
#### **常规**
- 物品栏内的方块现在是3D的。
- 缩短了渲染距离并出现了雾。
- 行走、破坏和放置方块的声音有问题。

## **触摸屏控制版本**
<div style="text-align:left"><img class="mw-file-element" src="https://zh.minecraft.wiki/images/thumb/Pocket_Edition_0.1.0.png/300px-Pocket_Edition_0.1.0.png?c991d" width="300" height="180" data-align="left" loading="lazy"></div>

2011年9月27日，一个标记为`minecraftpe.apk`的版本被上传到assets.minecraft.net的pe/目录。此版本与Xperia PLAY版本之间最显著的区别在于，它可以在普通的Android设备上运行，而且自带触摸屏控制。与正式发布版不同，该版本的Java代码没有混淆，这表明它可能是一个调试版本。

## **你知道吗**
- 如果在Android 3.0或更高版本上运行，游戏极有可能崩溃。
